Tipos primitivos de dados em Javascript

String

"string" é um dado valor do tipo caractere, ou seja, uma frase dentro de aspas, sejam aspas simples ou duplas ('', ""). Tudo o que estiver dentro de aspas, para o javascript isso é uma string. E além do mais, existe outra forma de escrever strings utilizando o sinal de cráse (``), com esse sinal de crase vc pode também escrever código javascript dentro dele, mas isso a gente verá depois. Ex: 

         const name = "Everson"
         const sobrenome = 'Vinicius'
         const petName = `Elisa`

Number

O tipo número é bastante simples de entender, pois ele possui um valor numérico e sinal, dependendo do seu valor. E com eles, naturalmente podemos fazer operações matemáticas. Coisa que não podemos fazer com strings. Ex: 

         const inteiro = 10
         const decimal = 3.14

Boolean

O tipo boolean é um tipo que possui apenas dois valores, verdadeiro ou falso. Ex: 

         const verdadeiro = true
         const falso = false

undefined

O tipo undefined é quando não temos um valor definido. Também pode ser considerado como "vazio". Geralmente acontece quando uma variável não tem nenhum valor ou alguma função que não retorna nada. É um valor que não foi definido, ou seja, não existe, undefined. 

         let meuValor
         console.log(meuValor) // retorna undefined

Null

Null acaba se comportando muitas vezes como undefined, mas para o javascript o "null" ele pode ser considerado um objeto. Ele também é considerado um valor inexistente. 

         const animal = null

Object

O tipo object no javascript ele é uma estrutura formada por propriedade e valor. Essas propriedades nós colocamos dentro de chaves. Outro detalhe importante, é que as propriedades dos objetos podem receber quaisquer tipos de valores, qualquer um dos que já vimos acima. 

         const Pessoa = {
            nome: "jessica",
            sobrenome: "de oliveira",
            casada: false,
            certidaoCasamento: undefined,
            bichinhosFavoritos: [
              "Sapos", 
              "Cachorros", 
              "Gato"
            ]
         }

Como acessar os valores das propriedades do nosso objeto?

Para acessar os valores do nosso objeto nós utilizamos o nome do nosso objeto, depois colocamos um ponto e em seguida o nome da nossa propriedade. Por exemplo, supondo que no nosso objeto "Pessoa", nós queremos acessar a propriedade "nome":

meuObjeto.nome_da_propriedade

        console.log(Pessoa.nome)
        // retorna → "jessica"

E se quisermos acessar a propriedade "sobrenome":

        console.log(Pessoa.sobrenome)
        // retorna → "de oliveira"

Podemos também alterar os valores do nosso objeto utilizando a mesma notação de verificar o valor da propriedade: 

        Pessoa.sobrenome = "Soares"
        console.log(Pessoa.sobrenome)
        // retorna → "soares"

Além de usar o ponto como uma forma de se alcançar o valor de uma propriedade do nosso objeto, também temos outra forma de buscar o valor de uma propriedade, passando o nome do nosso objeto, depois abre e fecha colchetes, e dentro dos colchetes colocamos o nome da propriedade dentro de aspas

meuObjeto["nome_da_propriedade"]

Exemplo:

        console.log(Pessoa["nome"])
        // retorna → "jessica"

Arrays

Arrays são nada mais que listas, e nessas listas podemos ter uma lista de absolutamente qualquer coisa. Podem ser uma lista de strings, de numbers, undefined e outros objetos. Podemos tanto ter uma lista de apenas um tipo de elementos, como uma lista de tipos variados e misturados. 

         const ListaDeCompras = [
          "pipoca", 
          'nescau', 
          `bolo`, 
           999, 
           true, 
           undefined, 
           null
          ]

Como acessar os valores das propriedades do nosso objeto?

Os arrays são listas e de tal forma, ele utiliza índices para indicar a posição dos seus elementos. Enquanto nos objetos a gente utilizava o nome do nosso objeto, depois um ponto e o nome da propriedade, no caso do array, utilizamos números.

Esses números que precisamos indicar são referentes a posição daquele elemento na minha lista.

Como acessar um elemento dentro do Array?

Para acessarmos um elemento específico da nossa lista, nós precisamos passar o seu índice, que é a sua posição no array. Vale ressaltar que o índice é um número inteiro, então nada de colocar string aí dentro, somente números e INTEIROS.

meuArray[indice]

Podemos observar que a forma de acessar um elemento do array é bem parecido com o que fazemos para acessar uma propriedade de um objeto, mas no caso do array, colocamos números dentro do array, e nos objetos colocamos uma string com o nome da nossa propriedade

Em javascript começa a se contar a partir de 0 em diante. Ou seja:

Se desejarmos pegar o primeiro elemento do nosso array, temos que escrever: 

        const listaDeCompras = ["leite", "chocolate", "açúcar"]
        listaDeCompras[0]
        // retorna → "leite"

        listaDeCompras[1]
        // retorna → "chocolate"

        listaDeCompras[2]
        // retorna → "açúcar"

E assim por diante. Também podemos adicionar alterar valores e adicionar valores ao nosso Array. Passamos uma posição da nossa lista e atribuimos um valor. Levando em consideração o nosso array "listaDeCompras". Exemplo: 

        listaDeCompras[0] = "suco"
        console.log(listaDeCompras)
        // retorna → ["suco", "chocolate", "açúcar"]

Além de passar um valor novo para um índice que estava preenchido, também podemos passar um valor para um índice que não tinha nada. O nosso array tinha três itens "suco", "chocolate" e "açúcar", ou seja, índices 0, 1 e 2, respectivamente. Podemos passar um novo elemento para o índice três, quatro, ou o índice que preferir: 

        listaDeCompras[3] = "pipoca"
        console.log(listaDeCompras)
        // retorna → ["suco", "chocolate", "açúcar", "pipoca"]

Agora vamos atribuir um novo elemento para um índice vazio, vamos tentar colocar um elemento num índice 5. Os índices 0, 1, 2, 3 já estão ocupados, pulamos o 4 e vamos direto para o 5 

        listaDeCompras[5] = "biscoito"
        console.log(listaDeCompras)
        // retorna → ["suco", "chocolate", "açúcar", "pipoca", undefined, "biscoito"]


        typeof listaDeCompras[4]
        // retorna → "undefined"

Notasse que agora que adicionamos algo no índice de número 5, o índice 4 foi criado e recebeu um valor indefinido, justamente porque não passamos nada para ele, e só existe um índice 5, se tiver um de número 4.

Como saber o tipo de um valor?

Existem algumas formas de se descobrir qual o tipo de um valor, a mais conhecida e a que funciona para a maior parte dos elementos javascript é o "typeof"

typeof

O typeof nos retornar qual o tipo daquele determinado valor e para utilizá-lo, podemos fazer assim: 

        const myName = "Everson"
        console.log(typeof myName)
        // Ele irá retornar → "string"

Outro exemplo:

        const idade = 23
        console.log(typeof idade)
        // Ele irá retornar → "number"

O typeof vai nos informar qual o tipo correto de STRING, NUMBER, BOOLEAN e UNDEFINED, mas ele não irá funcionar corretamente se tentarmos verificar qual o tipo de um objeto, ele nos retorna "object", o que faz sentido. Mas não funciona bem para verificar o tipo de um array, ele também nos retorna "object", e faz a mesma coisa com "null". 

        const lista = []
        console.log(typeof lista)
        // Ele irá retornar → "object"
      
        const meuObjeto = {}
        console.log(typeof meuObjeto)
        // Ele irá retornar → "object"
      
        const dinheiro = null
        console.log(typeof dinheiro)
        // Ele irá retornar → "object"

Como saber quando um elemento é um Array? e quando é Object?

Array.isArray()

O javascript possui uma função nativa que faz a verificação se um determinado valor inserido entre os parênteses é, ou não, um array. Ele nos retorna um valor true ou false, e com base nisso nós conseguimos saber se um determinado valor é um Array ou não. 

        const lista = []
        Array.isArray(lista)
        // Ele irá retornar → true
      
        const meuObjeto = {}
        Array.isArray(lista)
        // Ele irá retornar → false

meuObjeto instanceof Object

Para saber se um determinado valor é de fato um típico objeto javascript, utilizamos o método "instanceof" que faz a verificação se aquele determinado objeto que estamos querendo descobrir é ou não um objeto normal, com propriedades e valor. Assim como no método de verificação do Array, ele nos retorna um valor true ou false. 

        const meuObjeto = {name: "everson", age: 23}
        meuObjeto instanceof Object
        // Ele irá retornar → true
      
        const minhaLista = ["uva", "laranja"]
        minhaLista instanceof Object
        // Ele irá retornar → false

Note que se tivéssemos feito a verificação do "meuObjeto" e da "minhaLista" com o typeof, ele nos teria dito que ambos são object, o que de fato não é.
